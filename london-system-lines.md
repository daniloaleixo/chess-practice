# London System - 10 Most Famous Lines

The London's core idea is `d4 + Nf3 + Bf4 + e3 + Bd3` — a solid, low-theory setup that works against almost anything Black plays.

---

## 1. Classical Main Line
```
1. d4 d5 2. Nf3 Nf6 3. Bf4 e6 4. e3 Bd6 5. Bg3 O-O 6. Bd3
```

## 2. London vs King's Indian
```
1. d4 Nf6 2. Nf3 g6 3. Bf4 Bg7 4. e3 O-O 5. Be2 d6 6. O-O
```

## 3. London vs Queen's Indian
```
1. d4 Nf6 2. Nf3 e6 3. Bf4 b6 4. e3 Bb7 5. Bd3 c5 6. c3
```

## 4. London vs Dutch Defense
```
1. d4 f5 2. Nf3 Nf6 3. Bf4 e6 4. e3 d5 5. Bd3 Bd6 6. Bg3
```

## 5. London vs ...c5 (Anti-London)
```

```
